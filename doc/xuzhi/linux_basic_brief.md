# Linux 基础 学习小组

## 学习目标

了解linux基本原理和常用操作，为进一步提高应用水平打基础。

## 学习大纲
- 简介
  - 简史
  - 应用范围广泛
  - 命令行是基础的重中之重
  - 英语阅读能力至关重要
  - 虚拟机安装

- 基础命令
  - 文件、目录操作 cd/ls/cp/rm/mkdir/cat/more/less
  - 命令类型（内置、应用、脚本、别名）
  - 帮助信息
  - 应用程序安装 rpm/yum/repo配置/编译安装
  - 用户和权限
  - 文本编辑 vi/vim/nano
  - 压缩/解压缩 tar/cpio/gzip/zip/7z
  - 软硬链接
  - 环境配置 profile/bashrc
  - 系统监控 ps/top/netstat
  - 服务管理
  - 网络配置/防火墙
  - 磁盘管理

- Shell脚本编程

- 常见服务器应用
  - apache/nginx
  - tomcat/jetty/java app/php
  - mysql/mariadb
  - redis/memcached/mongodb
  - lvs/haproxy/nginx
  - samba
  - vsftp
  - dns
